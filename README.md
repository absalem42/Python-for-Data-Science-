# Python for Data Science

This repository contains progressive Python exercises focused on core programming and data science foundations.

## Repository structure

- `python_0_Strings/` - string handling and Python basics
- `python_1_Array/` - arrays, NumPy, and image manipulation
- `python_2_Datatable/` - CSV loading, pandas, and plotting
- `python_3_OOP/` - object-oriented programming exercises

Each folder is split into `ex00`, `ex01`, ... with standalone scripts.

## Requirements

- Python 3.10+
- Optional packages used by exercises:
  - `numpy`
  - `pandas`
  - `matplotlib`
  - `pillow`

Install packages with:

```bash
pip install numpy pandas matplotlib pillow
```

## Run examples

From the repository root:

```bash
python python_0_Strings/ex00/Hello.py
python python_1_Array/ex03/zoom.py
python python_2_Datatable/ex02/aff_pop.py
python python_3_OOP/ex00/tester.py
```
